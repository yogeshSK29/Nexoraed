# Nexoraed
Project Description

This is a web-based course website developed using Flask. The platform allows users to explore different courses, enroll in programs, and access learning content through an easy-to-use interface.

🚀 Features
📚 Display of multiple courses and services
🎥 Video-based learning content
🏷️ Different plans: Basic, Standard, Premium
💳 Online payment system (Razorpay & UPI)
📱 Responsive design for all devices
